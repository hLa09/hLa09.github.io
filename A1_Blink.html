Calculation:

Beside the LEDs and coding, there is a component called the resistors, here is the appropriate resistor value for each color LED was calculated as follow:

  • Red LED:
    • knowns: 5 Volt, desire LED at 20mA current, Voltage drop (Red) = 1.8V.
    • Formula: V = I(Amperes) * Resistance (Ohms)
    • 5V - 1.8V = 20mA * R
    • 3.2V = 20mA * R
    • R = 3.2 / 0.02 = 160 Ohms
  • Green LED:
    • knowns: 5 Volt, desire LED at 20mA current, Voltage drop (Green) = 1.8V.
    • Formula: V = I(Amperes) * Resistance (Ohms)
    • 5V - 1.8V = 20mA * R
    • 3.2V = 20mA * R
    • R = 3.2 / 0.02 = 160 Ohms
  • Yellow LED:
    • knowns: 5 Volt, desire LED at 20mA current, Voltage drop (Yellow) = 1.8V.
    • Formula: V = I(Amperes) * Resistance (Ohms)
    • 5V - 1.8V = 20mA * R
    • 3.2V = 20mA * R
    • R = 3.2 / 0.02 = 160 Ohms


For all of the LEDs: Red, Green, and Yellow require a resistor of around 160 Ohms, in which the closest resistor in my kit is resistor of 220 Ohms. As this resistor (220 Ohms) provide the closes resistances that each LEDs need, which I will have three of the 220 Ohms resistors on my circuit board to limit the currents Amperes passing through.

Code:

As mentioned there is a require function from the Arduino IDE to tell the LEDs how to behave to make the effects comes in to play. The following is a snippet of the Arduino code: 

          // Variable will not change
          // Declaring LED colored pin
          // LED represent for Red is connect to pin 2
          const int ledPinR = 2;
          // LED represent for Yellow is connect to pin 4
          const int ledPinY = 4;
          // LED represent for Green is connect to pin 7
          const int ledPinG = 7;

          // This act like the motor of the code, by running the code in order
          // the pinMode as output
          void setup() {
            // Set pin for LED Red as output
            pinMode(ledPinR, OUTPUT);
            // Set pin for LED Yellow as output
            pinMode(ledPinY, OUTPUT);
            // Set pin for LED Green as output
            pinMode(ledPinG, OUTPUT);
          }

          // This allow the following behavior to loop until changes
          void loop() {
            // Setting the Red on, while having the remain LED off
            digitalWrite(ledPinR, HIGH);
            // This LED yellow is off
            digitalWrite(ledPinY, LOW);
            // This LED green is off
            digitalWrite(ledPinG, LOW);
            // Delaying for 100 milliseconds before turn another light on
            delay(100);

            // Setting the Yellow on, while having the remain LED off
            // This LED red is off
            digitalWrite(ledPinR, LOW);
            // This LED yellow is on
            digitalWrite(ledPinY, HIGH);
            // This LED green is off
            digitalWrite(ledPinG, LOW);
            // Delaying for 100 milliseconds before turn another light on
            delay(100);

            // Setting the Green on, while having the remain LED off
            // This LED red is off
            digitalWrite(ledPinR, LOW);
            // This LED yellow is off
            digitalWrite(ledPinY, LOW);
            // This LED green is on
            digitalWrite(ledPinG, HIGH);
            // Delaying for 100 milliseconds before turn another light on
            delay(100);

            // Setting the Yellow on, while having the remain LED off
            // This LED red is off
            digitalWrite(ledPinR, LOW);
            // This LED yellow is on
            digitalWrite(ledPinY, HIGH);
            // This LED green is off
            digitalWrite(ledPinG, LOW);
            delay(100);
          }
