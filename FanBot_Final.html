Final Project:FanBot!

Introduction:

I want to build a robot that turn on fan when the surrounding becames hot/humidity, and display the temperature on the display screen. My ultimately goal is to be able to let use changes the speed of the fan when neccessary.

For this assignment I plan to have the following materials:

  • A Robot
    • A prototype robot for visualization, and project completeness.
  • LCD 1602 module
    • To display the temperature of the surrounding
  • DC motor and Fan
    • To have the fan turning when temperature/ or surrounding get hot/humidity.
  • LEDs indicators
    • The purpose of the LEDs is to display the temperature meter
    • For example: if the temperature cool it Blue, if it warm it's Yellow and while its hot it Red.
  • DHT11 Temperature + Humidity
    • I plan to use this to check for the surrounding temperature and humidity
  • Remote
    • My ultimate goal is to alos have user being able to change the speed of the fan, so I plan to have remote being implemented in the project.
    • For say if the user want to control from afar they could.
    • That means I would need an IR receiver to detect the code from the remote.

  • Second week: Finishing code and ensure all componet function properly,and to desire goal.
  • Third week: Finsihing up any remain unfinished and implementing low-mid profile of a robot prototype to complete the visualization.
