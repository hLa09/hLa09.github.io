Resistors:

Beside the 7-segment display and the capacitive sensor component, there is a component called the resistors, here is the appropriate resistor value for each resistors used:

  • 7-segments: This calcultion will be for per segments
    • Knowns: 5 Volt, desire LED at 20mA current, Voltage drop (per segments) = 2V.
    • Formula: V = I(Amperes) * Resistance (Ohms)
    • 5V - 2V = 20mA * R
    • 3V = 20mA * R
    • R = 3 / 0.02 = 150 Ohms
  • Capacitive Sensor:
    • Knowns: Capacitive sensor have multiple resistor choices (according to class lector notes)
    • Use a 1 megaohm resistor (or less maybe) for absolute touch to activate.
    • With a 10 megohm resistor the sensor will start to respond 4-6 inches away.
    • With a 40 megohm resistor the sensor will start to respond 12-24 inches away (dependent on the foil size). Common resistor sizes usually end at 10 megohm so you may have to solder four 10 megohm resistors end to end.


The number I got for the 7-segment display for voltage draw was from Farnell Datasheet found in the Arduino forum. The absolutemax forward current is the 25mA continuos, while at peak is 100mA. Although we are only looking at the continuos current flow, rather than at peak. also, the desire current flow is at 20mA, in which I use the 20mA current flow as a limit current. Looking at the graph the forward current per segment compare to the forward voltage, the range of the current is rise from 0 to 100mA is around 2V. Based on this information I assume the voltage drop will be at 2, and after calculation I ended with 150 ohms. The closest to the resistor I have is the 220 ohms resistors.

As for the Capacitive sensor, according to the notes and guidelines I decided to use 1M (megaohm) as my resistor for the sensing object. Because I would like to sense the number of time the object is being touched. [ Even though we will only be able to display the number from 0 to 9, when using one 7-segment display screen.]
